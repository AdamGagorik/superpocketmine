PocketMine
==========

This is a PocketMine server setup to run on a Linux server.

Usage
=====

#### download

```bash
./bin/download.sh
```

#### start

```bash
make start
```

#### stop

```bash
make attach
stop
```

#### attach

```bash
make attach
```

#### detach (while attached)

```bash
CTRL + a
d
```
